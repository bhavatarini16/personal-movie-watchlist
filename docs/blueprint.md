# **App Name**: CineTrack – Personal Movie Watchlist & Analytics App

## Core Features:

- User Authentication: Enable secure user registration, login with JWT, and management of user profiles for personalized access.
- Movie Discovery & Details: Search for movies using the TMDB API and view comprehensive details including posters, ratings, genres, release dates, overviews, and trailer links.
- Watchlist Management: Allow users to manage movies by adding them to a watchlist, marking them as watched, removing them, and adding personal ratings and notes.
- Cinema Personality Quiz: An interactive quiz that uses a tool to classify the user into a specific cinema personality type (e.g., Action Addict, Sci-Fi Thinker) based on their movie preferences.
- Analytics Dashboard: A personalized dashboard displaying various movie statistics, such as total movies watched, favorite genres, average ratings, monthly activity, and top actors, with visual charts.
- Friend Compatibility Analyzer: Compare movie tastes with friends, showing similar watched movies, genre similarity, a compatibility score, and using a tool to suggest movies both users might enjoy.
- Smart Movie Recommendations: An intelligent tool that suggests movies based on the user's watch history, favorite genres, and given ratings.

## Style Guidelines:

- Dark cinematic theme with black as the base color, complemented by deep red or deep purple gradient accents.
- Use a sophisticated font for headlines and a highly readable font for body text, maintaining a premium feel.
- Modern, crisp icons that align with the cinematic and glassmorphism aesthetic.
- Immersive, movie-first layout prioritizing large-scale movie posters, backdrops, and visual storytelling, including a grid layout for movie posters.
- Glassmorphism-style UI components with subtle blur and depth effects.
- Fully responsive design optimized for seamless experience across mobile and desktop devices.
- Smooth animations and fluid transitions throughout the interface, including subtle micro-interactions on hover and elegant page transitions.
